# Dad Joke DB server

## Description

This is the backend for [Dad Joke DB](https://marcuses101.github.io/dad-jokes).
Fetches either a random joke or a joke based on a search term. The joke is processed by Google Natural Language Processing API to extract relevant entities. The entity is used to find a relevant clipArt Image using Bing image search. Data is sent back to the client.

## Tech

- Node/ Express
- Google Natural Language Processing API
- Azure Cognitive Services Bing Image Search
